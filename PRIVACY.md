# PII Shield Privacy Policy

**Stateless operation:** PII Shield processes data in-memory. We do not store, log, or transmit the content of your Jira comments to any permanent storage.

**Data retention:** Once a redaction is performed, the text is immediately purged from our server memory.

**Analytics:** We only store anonymous counters (e.g., "Total Redactions") to provide you with security metrics.

**Contact:** Support is provided via GitHub Issues at this repository.
